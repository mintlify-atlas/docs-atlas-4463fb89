## Use the live app

FretFlow runs entirely in your browser. No sign-up, no install.

<Card title="Open FretFlow" icon="arrow-up-right-from-square" href="https://iecg.github.io/fretboard-app/">
  Launch the live app at iecg.github.io/fretboard-app
</Card>

## Run it locally

If you want to develop or explore the source code:

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/iecg/fretboard-app.git
    cd fretboard-app
    ```
  </Step>
  <Step title="Install dependencies">
    Requires Node.js 20.12 or higher.
    ```bash
    npm install
    ```
  </Step>
  <Step title="Start the development server">
    ```bash
    npm run dev
    ```
    Open `http://localhost:5173` in your browser.
  </Step>
</Steps>

## Tour the interface

### 1. Pick a key

The **Circle of Fifths** in the Key panel is your root note selector. Click any segment to change the key. The center of the circle shows the root note and its key signature (e.g., `3♯` for A major).

Use the **♭ / ♯** toggle button next to the circle to switch between flat and sharp accidental notation.

### 2. Choose a scale

Click the **Scale** dropdown to pick from 12 scales, organized by type:

- Major modes: Major, Lydian, Mixolydian
- Minor modes: Natural Minor, Dorian, Phrygian, Locrian
- Harmonic: Harmonic Minor
- Pentatonic: Minor Pentatonic, Major Pentatonic
- Blues: Minor Blues, Major Blues

The fretboard updates immediately to highlight the selected scale.

### 3. Add a chord overlay

Click the **Chord Overlay** dropdown to superimpose a chord over the scale. Chord tones appear with a distinct color, while non-chord scale notes are dimmed.

Check **Link chord root to scale** to keep the chord root tied to the selected key, or uncheck it to set a different chord root.

### 4. Switch fingering patterns

In the **Settings** panel, choose one of three fingering views:

| Pattern | What it shows |
|---|---|
| All | Every note of the scale across the entire neck |
| CAGED | Individual or multi-select CAGED shape regions |
| 3NPS | 3-notes-per-string positions (1–7) |

### 5. Adjust the view

Use the **toolbar above the fretboard** to:

- **Frets** — narrow the visible fret window using the ◀ ▶ controls
- **Zoom** — increase fret column width beyond auto-fit
- **Go to** — jump to Open (fret 0), Mid (fret 5), or High (fret 12) positions

Drag the fretboard left and right to scroll when it overflows the container.

### 6. Play notes

Tap any highlighted note bubble to hear it played. Use the **mute button** (🔊) in the header to toggle audio on and off.

<Tip>
All settings — scale, key, fingering pattern, fret range, and more — are automatically saved to `localStorage` and restored when you return to the app.
</Tip>

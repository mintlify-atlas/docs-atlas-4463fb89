FretFlow is a browser-based tool for guitarists and music theory students. It visualizes scales, chords, and fingering patterns on an interactive fretboard — with no installation required.

## What FretFlow does

<CardGroup cols={2}>
  <Card title="Fretboard visualization" icon="table-columns" href="/features/fretboard">
    Highlight any scale across the full 24-fret neck, with a 3-tier note system that distinguishes chord tones, scale-only notes, and off-scale chord tones.
  </Card>
  <Card title="Scale & chord overlay" icon="layer-group" href="/features/scales-chords">
    Choose from 12 scales and 7 chord types. The chord overlay works independently from the scale root, so you can analyze chord tones against any scale.
  </Card>
  <Card title="CAGED shapes" icon="shapes" href="/features/caged-system">
    View individual or multiple CAGED shapes with color-coded backgrounds. Shift-click to multi-select. Works for all scales, not just pentatonics.
  </Card>
  <Card title="3 Notes Per String" icon="music" href="/features/3nps">
    Explore all 7 positions of any scale using the 3NPS fingering system, ideal for building speed and connecting positions.
  </Card>
  <Card title="Circle of Fifths" icon="circle" href="/features/circle-of-fifths">
    Interactive annular selector for the root note, with key signature display, scale degree labels, and enharmonic equivalents.
  </Card>
  <Card title="Audio playback" icon="volume-high" href="/features/fretboard#audio-playback">
    Tap any note to hear it played through the Web Audio API synth.
  </Card>
</CardGroup>

## Tech stack

FretFlow is built with:

- **React 19** + **TypeScript 6**
- **Vite 8** for bundling and dev server
- **Jotai** for state management (persisted to `localStorage`)
- **Lucide React** for icons
- **Web Audio API** for note synthesis
- Deployed to **GitHub Pages**
